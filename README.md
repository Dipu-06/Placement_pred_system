# 🎓 Student Placement Prediction System using Artificial Neural Network (ANN)

## 📖 Overview

This project is a **Student Placement Prediction System** built using **Deep Learning**. The model uses an **Artificial Neural Network (ANN)** to predict whether a student is likely to be placed based on various academic and personal features.

The project demonstrates the complete deep learning workflow, including data preprocessing, feature scaling, model building, training, evaluation, and prediction.

---

## 🚀 Features

* Data preprocessing and cleaning
* Feature scaling using StandardScaler
* Artificial Neural Network (ANN) for binary classification
* Model training and validation
* Performance evaluation
* Predict placement for new student data

---

## 🛠️ Tech Stack

* Python
* TensorFlow
* Keras
* NumPy
* Pandas
* Matplotlib
* Scikit-learn
* Jupyter Notebook

---
Workflow

1. Load the dataset
2. Perform data preprocessing
3. Split the data into training and testing sets
4. Scale numerical features
5. Build the Artificial Neural Network (ANN)
6. Train the model
7. Evaluate model performance
8. Predict student placement

---

## 🧠 ANN Architecture

* Input Layer
* Hidden Layer 1 (ReLU)
* Hidden Layer 2 (ReLU)
* Output Layer (Sigmoid)

**Loss Function:** Binary Crossentropy

**Optimizer:** Adam

**Evaluation Metric:** Accuracy
